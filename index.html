RACE (Resource Aware Cost- Efficient Scheduler) :

Description :

For my final year project, I worked on RACE (Race Resource Aware Cost Efficient Scheduler for Cloud-Fog Environments), which aimed to address the challenges of efficiently managing resources in hybrid cloud-fog computing environments. The cloud-fog paradigm combines the scalability and storage capabilities of cloud computing with the low-latency, localized processing of fog computing. This hybrid approach is becoming increasingly relevant with the rise of IoT devices, where processing needs to occur closer to the edge of the network. The main objective of RACE was to design and implement a scheduling algorithm that could optimize resource utilization while minimizing costs in such environments.
This involved several key features:
1. Resource Awareness: The scheduler intelligently allocates tasks to resources based on their capacity and availability, considering both cloud and fog nodes. This ensures that tasks are processed efficiently without overloading any single resource.
2. Cost Efficiency: By taking into account the operational costs of cloud and fog resources, the scheduler minimizes costs by choosing the most cost-effective resource for executing a task, whether it's on the cloud or fog infrastructure.
3. Dynamic Task Scheduling: RACE adapts in real-time to changes in resource availability and workload demands, ensuring that the scheduling decisions are always optimized for the current environment.
4. Latency Considerations: Since fog computing nodes are closer to end devices, they can handle tasks with low latency, which is crucial for time-sensitive applications. RACE accounts for these latency constraints when making scheduling decisions.
5. Scalability: The system was designed to scale with an increasing number of devices and workloads, which is essential in modern IoT-driven environments.
The final solution was implemented using a combination of simulation tools and real-world scenarios, and I evaluated its performance based on factors like resource utilization, task completion time, and operational costs. The results showed that RACE was able to significantly improve the overall efficiency of task scheduling in cloud-fog environments, making it a promising approach for future IoT-based applications. This project gave me hands-on experience in cloud and fog computing, resource management, and optimization algorithms, and helped me understand the real-world challenges in building scalable and cost-effective computing systems
